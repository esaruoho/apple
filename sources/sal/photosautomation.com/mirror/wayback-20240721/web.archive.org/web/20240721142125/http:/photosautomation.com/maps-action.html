The “Show Location in Maps” Action

The “Show Location in Maps” action for Photos will display a map, in the OS X Maps application, for a single image whose reference is passed as input to this action.

The action has parameters for setting the type of map to be displayed, the zoom level of the view, and whether a pin should be dropped at the image’s location.
