▲ Accessibility Commands

Commands for retrieving information about documents, slides, and slide items.

Assist with Presenter Notes
Display a floating window for entering, dictating, and editing presenter notes.
Stop Edit and Lock Item
Exit the current edit and lock the text item
Lock Text Items
Lock all text items on the current slide.
Lock Default Body Item
Lock the default body item if it is showing
Lock Default Title Item
Lock the default title item if it is showing
Read the Slide
This Dictation Command will speak the slide’s title, body, and image descriptions. Missing items are ignored. Also, the contents of text items will be read if no title or body is showing.

NOTE: specific sounds are played before each reading indicating the type of element whose contents are about to be read. The higher sound (F/C3) indicates an image description is about to be read; the middle sound (D/C3) indicates the title is about to be read; and the lower sound (G/C2) indicates that the body is about to be read. If the contents of text items are read (see above), a secondary middle tone (C/C3) will be sounded.
Read Text of Current Text Box
Have the computer speak the text in the current text box
Read the Next Slide
This Dictation Command will perform a slide advance, and if the next slide is displayed, it will then speak the new slide’s title, body, and image descriptions. Missing items are ignored. (see the “Read this Slide” command for more deatils)
Speak the body of current slide
Speak the content of the default body text of the current slide.
Count of the Current Slide’s Items
Speak information about how many of each class of elements are on the current slide. e.g. Shapes, Lines, Movies, Text Items, Images, etc.
Info about the Current Slide’s Items
Speak information about each element on the current slide
Info about each image of current slide
Speak information about each of the images on the current slide.
Info about the Current Slide
Info about the Current Slide
Speak the title of current slide
Speak the title of the current slide.
Speak the title and body of current slide
Speak the content of the default title and body text of the current slide.
Info about the Current Slide’s Transition
Speak information about the transition of the current slide
Info about the Frontmost Presentation
Speak information about the frontmost document
Unlock Text Items
Unlock all text items on the current slide.
Unlock Default Body Item
Unlock the default body item if it is showing
Unlock Default Title Item
Unlock the default title item if it is showing
