▲ Slide: Image(s)

Dictation Commands for working with images on the current slide of the frontmost Keynote presentation

Choose File for First Image of Current Slide
Using the standard system chooser dialog, select and import an image file for the first image of the current slide
Choose File for Second Image of Current Slide
Using the standard system chooser dialog, select and import an image file for the second image of the current slide
Choose File for Third Image of Current Slide
Using the standard system chooser dialog, select and import an image file for the third image of the current slide
Choose File for Fourth Image of Current Slide
Using the standard system chooser dialog, select and import an image file for the fourth image of the current slide
Choose File for Fifth Image of Current Slide
Using the standard system chooser dialog, select and import an image file for the fifth image of the current slide
Choose File for Sixth Image of Current Slide
Using the standard system chooser dialog, select and import an image file for the sixth image of the current slide
Choose File for the Selected Image
Using the standard system chooser dialog, select and import an image file for the selected image. If no image is selected, the image on the top of the image stack will be used.
Choose Files for All Images on Current Slide
Using the standard system chooser dialog, select and import an image file for each of the images on the current slide
Choose Files for the Selected Images
Using the standard system chooser dialog, select and import an image file for each of the selected images. If no images are selected, then all images on the slide will be processed.
Edit Selected Image in Photos
Edit the selected image in the Photos application. NOTE: the selected image must be in the current Photos library, and have been imported into the presentation using Dictation Commands.
Import Selected Finder Item
Import the currently selected image file in the Finder into the selected image frame in Keynote.

If no image is selected in Keynote, and the slide contains a single image, it will be the recipeient of the import.

If the current slide contains multiple images, the selected image will be the recipient of the import.

If the slide has no images, the Finder item will be imported into the slide and automatically sized.

NOTE: Imported image files will include any embedded description metadata. Also, movie files are compatible for import and will be replace the contents of a target Keynote image frame.

Import Selected Photos Item
Import the currently selected photo in Photos into the selected image in Keynote.

If no image is selected in Keynote, and the slide contains a single image, it will be the recipeient of the import.

If the current slide contains multiple images, the selected image will be the recipient of the import.

If the slide has no images, the exported Photos photo will be imported into the slide and automatically sized.

NOTE: imported images are automatically “linkable” back to Photos, and will include any description from Photos.

Overlay Image Descriptions
Place the description of each image on top of it as a text overlay.
Overlay Image Titles
Place the title (name) of each image on top of it as a text overlay. If the image was previously imported into Keynote from Photos using Dictation Commands, this command will attempt to locate the image in the current Photos library and use the title assigned to it.
Add a QR code using contents of selected text item
This command will convert the contents of the selected text item into a QR code and add it to the current slide. NOTE: select the text item, not its contents.
Scale Image to Fit Slide Height
Scale the selected image to fit the slide height and then center the scaled image horizontally.
Scale Image to Fit Slide Width
Scale the selected image to fit the slide width and then center the scaled image vertically.
Show Selected Image in Photos
Show the selected image in the Maps application. NOTE: the selected image must be in the current Photos library, and have been imported into the presentation using Dictation Commands.
Show Selected Image in Photos
Show the selected image (full-window) in the Photos application. NOTE: the selected image must be in the current Photos library, and have been imported into the presentation using Dictation Commands.
Describe Largest Image
Speaks the description of the largest image (by volume) on the current slide.
Update Selected Image with Photos Master
Update the selected image with a new export of its related master image from the current Photos library. NOTE: the selected image must be in the current Photos library, and have been imported into the presentation using Dictation Commands.
