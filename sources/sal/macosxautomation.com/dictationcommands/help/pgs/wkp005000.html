▲ Finder Locations

Dictation Commands for opening designated folders.

Open Applications Folder
This command will switch to the Finder and open the Applications folder.
Open Desktop Folder
This command will switch to the Finder and open the current user’s Desktop folder.
Open Documents Folder
This command will switch to the Finder and open the current user’s Documents folder.
Open Home Folder
This command will switch to the Finder and open the current user’s home directory.
Open User iCloud Drive Folder
This command will switch to the Finder and open the current user’s iCloud Drive folder.
Open Computer Library Folder
This command will switch to the Finder and open the computer’s Library folder.
Open Movies Folder
This command will switch to the Finder and open the current user’s Movies folder.
Open Music Folder
This command will switch to the Finder and open the current user’s Music folder.
Open Pictures Folder
This command will switch to the Finder and open the current user’s Pictures folder.
Open Public Folder
This command will switch to the Finder and open the current user’s Public folder.
Open Shared Items Folder
This command will switch to the Finder and open the Shared Items folder.
Open User Applications Folder
This command will switch to the Finder and open the current user’s Applications folder.
Open User Library Folder
This command will switch to the Finder and open the current user’s Library folder.
