▲ Media-Item: Utilities

Dictation Commands for working with media items.

Crop Selected Photo
Enter the crop edit mode for the currently selected photo.
Edit Selected Photo
Edit the selected media item.
Exit Edit Mode
Leave edit mode for the current image and return to albums or photos view.
Select All Photos
Select all of the media items in the Photos window. Equivalent to typing Command-A.
Show this Image in Keynote
Switch to Keynote and show the slide containing the currently selected media item.
