▲ Move Selected Finder Items with Replacement into user’s Applications folder

Description: Move the items selected in the Finder into the user’s Applications folder, replacing existing items with the same name. The moved items will highlighted after moving.
Scope: Finder
Command Variations: To use this command, speak any of the following phrases. Words within brackets [ ] are optional and my be omitted if desired.
  • move this with replacement to my applications [folder]
  • move this with replacement into my applications [folder]
  • move these with replacement to my applications [folder]
  • move these with replacement into my applications [folder]
