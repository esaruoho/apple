▲ Presentation: Metadata

Dictation Commands dealing with the metadata of a presentation.

Show Finder Info window of Front Presentation File
Show Finder Info window for the file of the frontmost presentation.
Show File of Front Presentation
Show the file of the frontmost presentation in the Finder.
Show Font List
This command will display a list of the typefaces used in the frontmost presentation.
Show Related Items
This command will display a Spotlight search window in the Finder showing items whose tags are related to the frontmost presentation.
