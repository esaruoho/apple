▲ Overlay Image Titles

Description: Place the title (name) of each image on top of it as a text overlay. If the image was previously imported into Keynote from Photos using Dictation Commands, this command will attempt to locate the image in the current Photos library and use the title assigned to it.
Scope: Keynote
Command Variations: To use this command, speak any of the following phrases. Words within brackets [ ] are optional and my be omitted if desired.
  • overlay image titles
  • overlay image names
  • put titles over each image
  • put names over each image
  • put titles on top of every image
  • put names on top of every image
