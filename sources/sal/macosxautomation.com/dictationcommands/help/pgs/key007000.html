▲ Slide: Master

Dictation Commands for changing the master slide of the current slide of the frontmost Keynote presentation

Set Master Slide to “Blank”
Changes the master slide of the current slide to “Blank”
Set Master Slide to “Bullets”
Changes the master slide of the current slide to “Bullets”
Choose New Master Slide from List
This command will display the names of the available master slides. Select a name to change the master slide of the current slide.
Set Master Slide to “Photo”
Changes the master slide of the current slide to “Photo”
Set Master Slide to “Photo - Horizontal”
Changes the master slide of the current slide to “Photo - Horizontal”
Set Master Slide to “Photo - Horizontal Alt”
Changes the master slide of the current slide to “Photo - Horizontal Alt” (only in the Photo Essay theme)
Set Master Slide to “Photo - 3 Up”
Changes the master slide of the current slide to “Photo - 3 Up”
Set Master Slide to “Photo - Vertical”
Changes the master slide of the current slide to “Photo - Vertical”
Set Master Slide to “Quote”
Changes the master slide of the current slide to “Quote”
Set Master Slide to “Quote Photo”
Changes the master slide of the current slide to “Quote Photo” (only in Photo Essay theme)
Set Master Slide to “Title & Bullets”
Changes the master slide of the current slide to “Title & Bullets”
Set Master Slide to “Title, Bullets & Photo”
Changes the master slide of the current slide to “Title, Bullets & Photo”
Set Master Slide to “Title - Center”
Changes the master slide of the current slide to “Title - Center”
Set Master Slide to “Title & Subtitle”
Changes the master slide of the current slide to “Title & Subtitle”
Set Master Slide to “Title - Top”
Changes the master slide of the current slide to “Title - Top”
