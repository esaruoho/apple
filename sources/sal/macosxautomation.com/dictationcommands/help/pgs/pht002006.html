▲ Change Export Copyright String

Description: This command will display a text input dialog in which you enter the copyright string to use to overlay exported images. Once a string has been stored, it will be used without prompting until this command is given t change it.
Scope: Photos
Command Variations: To use this command, speak any of the following phrases. Words within brackets [ ] are optional and my be omitted if desired.
  • change [the] export copyright
  • set [the] export copyright
