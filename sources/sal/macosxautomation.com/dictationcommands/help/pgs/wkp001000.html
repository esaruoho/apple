▲ Computer Camera

Dictation Commands for controlling the computer’s camera

Take My Picture
Display the system picture taker window for the computer’s camera. Once the user has taken a snapshot, the file will be imported into the current Photos library.
Take My Picture Automatically
Display the system picture taker window for the computer’s camera and take the snapshot automatically after a coutndown. Once snapshot has been taken, the file will be imported into the current Photos library.
