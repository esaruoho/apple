▲ Speak Slide Number

Description: This command will speak the slide number of the current slide
Scope: Keynote
Command Variations: To use this command, speak any of the following phrases. Words within brackets [ ] are optional and my be omitted if desired.
  • what slide is this
  • what is [the] number of this slide
  • speak [the] slide number
  • [say] [the] slide number of this slide
