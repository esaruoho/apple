▲ Export Table to Keynote as Pie Chart

Description: Export the data of the selected table row or column to a new slide in the frontmost Keynote presentation, as a Pie chart.
Scope: Numbers
Command Variations: To use this command, speak any of the following phrases. Words within brackets [ ] are optional and my be omitted if desired.
  • export [this] row to keynote as [a] pie chart
  • export [the] selected row to keynote as [a] pie chart
  • export [this] column to keynote as [a] pie chart
  • export [the] selected column to keynote as [a] pie chart
