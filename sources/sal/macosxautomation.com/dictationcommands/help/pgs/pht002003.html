▲ Export to File as Sequence

Description: Export the selected media items to files named sequentially using the user-provided base name. The media items are exported in high-quality JPEG format.
Scope: Photos
Command Variations: To use this command, speak any of the following phrases. Words within brackets [ ] are optional and my be omitted if desired.
  • export these as [a] sequence
  • export these named in sequence
  • export these named sequentially
