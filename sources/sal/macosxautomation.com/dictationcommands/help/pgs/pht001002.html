▲ Clear All Titles and Descriptions

Description: After user approval, this command will clear all titles and descriptions from the selected media items.
Scope: Photos
Command Variations: To use this command, speak any of the following phrases. Words within brackets [ ] are optional and my be omitted if desired.
  • clear [all] titles and descriptions
  • remove [all] titles and descriptions
  • clear every title and description
  • remove every title and description
