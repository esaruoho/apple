▲ Media Item: Metadata

Dictation Commands for working with media item metadata.

Add Rating Keyword ★
This Dictation Command will apply a rating keyword of "★" to the selected media items. Any existing rating keyword will be removed.
Add Rating Keyword ★★
This Dictation Command will apply a rating keyword of "★★" to the selected media items. Any existing rating keyword will be removed.
Add Rating Keyword ★★★
This Dictation Command will apply a rating keyword of "★★★" to the selected media items. Any existing rating keyword will be removed.
Add Rating Keyword ★★★★
This Dictation Command will apply a rating keyword of "★★★★" to the selected media items. Any existing rating keyword will be removed.
Add Rating Keyword ★★★★★
This Dictation Command will apply a rating keyword of "★★★★★" to the selected media items. Any existing rating keyword will be removed.
Remove Rating Keyword
This Dictation Command will remove all rating keywords from the selected media items.
Search for Rating Keyword ★
This Dictation Command will search for media items with a rating keyword of "★"
Search for Rating Keyword ★★
This Dictation Command will search for media items with a rating keyword of "★★"
Search for Rating Keyword ★★★
This Dictation Command will search for media items with a rating keyword of "★★★"
Search for Rating Keyword ★★★★
This Dictation Command will search for media items with a rating keyword of "★★★★"
Search for Rating Keyword ★★★★★
This Dictation Command will search for media items with a rating keyword of "★★★★★"
Install Ratings Keywords
This Dictation Command will install these ratings keywords: "★", "★★", "★★★", "★★★★", "★★★★★"
Assist With Adding Descriptions
In turn, each of the selected photos will be shown full-width and a dialog will prompt you to enter the description for the currently dispalyed image.
Assist With Adding Titles
In turn, each of the selected photos will be shown full-width and a dialog will prompt you to enter the title for the currently dispalyed image.
Clear All Keywords
After user approval, this command will clear all keywords from the selected media items.
Clear All Titles and Descriptions
After user approval, this command will clear all titles and descriptions from the selected media items.
Show Locations in Maps
Show the locations of the selected media items in the Maps application.
