▲ Import Selected Finder Item

Description: Import the currently selected image file in the Finder into the selected image frame in Keynote.

If no image is selected in Keynote, and the slide contains a single image, it will be the recipeient of the import.

If the current slide contains multiple images, the selected image will be the recipient of the import.

If the slide has no images, the Finder item will be imported into the slide and automatically sized.

NOTE: Imported image files will include any embedded description metadata. Also, movie files are compatible for import and will be replace the contents of a target Keynote image frame.

Scope: Keynote
Command Variations: To use this command, speak any of the following phrases. Words within brackets [ ] are optional and my be omitted if desired.
  • import [selection] from [the] finder
  • import [the] finder selection
  • import [selected] item from [the] finder
  • import [selected] image from [the] finder
  • import [selected] photo from [the] finder
  • import [selected] movie from [the] finder
