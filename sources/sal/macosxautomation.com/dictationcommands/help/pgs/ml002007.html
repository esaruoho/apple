▲ Move to Junk Mailbox

Description: Move the selected message to the Junk folder.
Scope: Mail
Command Variations: To use this command, speak any of the following phrases. Words within brackets [ ] are optional and my be omitted if desired.
  • move this [message] to [the] junk [folder]
  • move these [messages] to [the] junk [folder]
  • junk this [message]
  • junk these [messages]
