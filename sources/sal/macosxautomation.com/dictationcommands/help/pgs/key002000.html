▲ Presentation: Manage

Dictation Commands for managing open Keynote presentations

Close Presentation: All-No Save
Close all open presentations without saving them
Close Presentation: Front-No Save
Close the frontmost presentation without saving it
Save Presentation: To Document Folder with Auto-Naming
Save the frontmost presentation to the Documents folder using the title of the first slide as the filename. If the frontmost presentation has already been saved, then save to existing file.
Layout View: Scale to Selected Items
Scale the layout view (up to 400%) to fit the selected slide items
Layout View: Scale to Slide
Scale the layout view to fit the current slide
