▲ Document: Template Assist

Provides interactive assistance to the user for entering the text and choosing the images in a template. The supported templates are: Tab Flyer, For Rent Flyer, For Sale Flyer, Lost & Found Flyer, Classic Certificate, Kids Certificate, Photo Poster Small, Photo Poster Big, Event Poster Small, Event Poster Big, Party Poster Small, Party Poster Big, School Poster Small, School Poster Big, Type Poster Small, and Type Poster Big.

Template Assist: Classic Certificate
This command will open a new copy of the Classic Certificate template and interactively prompt you to enter text and choose image files to replace the placeholders in the new document.
