▲ Displaying Albums

Dictatation Commands for Displaying Specified Albums. NOTE: many of the commands from this topic require the use of Photos v2.0 or higher.

Show All Albums
Show the icons for all albums in the main window.
Show Favorites Album
Show the contents of the Favorites album in the main window.
Show Last Import Album
Show the contents of the Last Import album in the main window.
Show Shared Items
Show all of the shared items in the main window.
