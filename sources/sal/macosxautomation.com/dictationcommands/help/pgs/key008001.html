▲ Export All Notes to Mail

Description: Create a new outgoing Mail message containing the presenter notes of the front presentation.
Scope: Keynote
Command Variations: To use this command, speak any of the following phrases. Words within brackets [ ] are optional and my be omitted if desired.
  • add [these] [presenter] notes to [a] [new] mail message
  • put [these] [presenter] notes in [a] [new] mail message
  • export [these] [presenter] notes to mail
