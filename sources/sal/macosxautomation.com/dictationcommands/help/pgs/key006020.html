▲ Import Selected Photos Item

Description: Import the currently selected photo in Photos into the selected image in Keynote.

If no image is selected in Keynote, and the slide contains a single image, it will be the recipeient of the import.

If the current slide contains multiple images, the selected image will be the recipient of the import.

If the slide has no images, the exported Photos photo will be imported into the slide and automatically sized.

NOTE: imported images are automatically “linkable” back to Photos, and will include any description from Photos.

Scope: Keynote
Command Variations: To use this command, speak any of the following phrases. Words within brackets [ ] are optional and my be omitted if desired.
  • import [selection] from photos
  • import [selected] item from photos
  • import [selected] image from photos
  • import [selected] photo from photos
  • import photos selection
