Workshop Session Summary

Congratulations. You’ve completed the process of creating digital publications using the automation tools of OS X.

Specifically:

  1. You installed a set of Automator service workflows that use text selected in documents or windows as input for Text to EPUB action in OS X.
  2. Using the newly installed services, you created both a simple formatted digital document, and a multi-chapter book containing images and audio files.

In short, you’ve discovered that the power and abilities of automation in OS X is available for you use. If you find yourself thinking “I wonder if there’s a better way to do this?” then it’s time to consider using automation.

Segment Overview

Engage the attendees in conversation about the workshop, lesson learned, ideas for other uses.

Talking Points

  1. Services created with Automator can be adapted, expanded, simplified, etc. To edit an existing service, choose “Open” from the Automator “File” menu, and then select “Service” from the “Type” popup menu in the forthcoming dialog. The services installed in the user Services folder will be selectable for editing.
  2. Services created with Automator can be shared with others. Mention how the service workflow files, located in the Home > Library > Services folder can be copied to other computers, and then installed by double-clicking the workflow files in the Finder.
