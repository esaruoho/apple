Talking Points

  1. Ask the attendees if they found the process of creating EPUB documents with the Text to EPUB services useful, and easy to accomplish.
  2. Discuss possible scenarios in which these services could be used, such as converting existing documents to digital format and for quick publishing of scholarly papers.
  3. For those attendees desiring greater customization of the created EPUB documents, point out that EPUB documents can be “unpacked”, edited, and “re-packed” using the script tools provided below. Custom CSS styles and media placement can be applied to the EPUB publications using standard HTML editing applications, such as TextWrangler or BBEdit.

EDITING EPUB DOCUMENTS

Essentially, EPUB documents are saved as archived folders containing editable files and media.

To make changes to an EPUB document, first drag it onto the Unpack EPUB dropet and an unpacked copy of the digital book will be placed on the desktop.

To re-pack the edited EPUB folder, launch the Re-Package EPUB script and choose the edited folder to be reconstituted and a repackaged EPUB document will be created.
