Viewing in iBooks on iOS Devices

For your convienience, you can DOWNLOAD the finished example EPUB document.

There are multiple options for transfering the created EPUB publication to an iPhone or iPad for viewing in iBooks.

  1. One option is to connect to the devices using iTunes and transfer the created publication file to the connected devices.
  2. Another option is to include the created files as attachments to messages sent to the mail accounts on your iOS devices. Tap and hold on the attachment and choose “Open in iBooks” from the forthcoming contextual menu.
  3. To view from another device like an iPad, you can click THIS LINK to send yourself an email message that contains the link to the finished EPUB document.
  4. However, an easier option is to view this webpage in Safari on the iPad or iPhone, and then tap this link: MAD TEA PARTY EPUB

    5. In the forthcoming menu, tap the Open in “iBooks” button:

Once the created EPUB file has been transfered to iBooks, it will appear on the bookshelf in the same manner as other digital publications. Open the book by tapping its cover image displayed on the bookshelf:

(⬆ see above ) The newly created book appears on the iBooks bookshelf.

The book will open to its table of contents, which in the case of a single-chapter book, is empty. Tap the Resume button at the top left of the window to begin reading the book:

(⬆ see above ) Since the created book only has one chapter, there is no table of contents. Tap the Resume button at the top left (outlined here in red) to begin or continue reading the book.

Note that although you can assign a different typeface and size to the text, the color, style, spacing, and alignment are the same as the original text from the webpage:

(⬆ see above ) Since the source text for the digital book was in Rich Text format, the text color, style, and alignment properties are retained. In iBooks, the font and type size can be set by the user.

WORKSHOP LEADER NOTES

Segment Overview

In this segment, attendees will view the EPUB document created in the previous segment. This can be accomplished by:

  1. Viewing the document on the Mac using a third-party application such as Bookle; or
  2. Viewing the document on an iPad or iPhone by opening this webpage on the device, and then tapping the MAD TEA PARTY EPUB link in the segment instructions. Doing so will trigger a download approval dialog on the device. Once approved, the document will be downloaded and opened in iBooks. Show the attendees how to send themselves a link to this page using Safari’s Mail Link to This Page command in the File menu. They can then tap the link in Mail on the iPad or iPhone to go to the webpage.

Talking Points

  1. As this tutorial demonstrates, using the automation tools in OS X, it is easy to convert text selected in documents, even webpages, into digital publications.
  2. Note that formatting, such as text styles (bold, italic, etc.), color, and tabbing, can be preserved using the action’s settings for honoring RTF (Rich Text Format) input. However, when viewed in iBooks, font family, and font sizing can still be adjusted by the reader.
  3. Discuss the use of the Text to EPUB services in conjunction with the Reader feature of Safari.

Speaker Note

The Text to EPUB action does not support RTFD (Rich Text with Images/Media) text selections or documents used as input. As a work-around, select the text separately from the included media, and then download and add the media using the input options described in the next workshop segment.
