HOW TO INSTALL

Installation of the Automator-based services is easy, just follow these steps:

  1. Download and upack the zip archive containing the Automator service workflow files.
  2. Open the folder containing the services, and double-click one of the service workflow files.
  3. A Service Installer dialog will display, prompting you to either open the workflow file in Automator or to approve the installation of the service. Click the Install button to install the service. The service will automatically be placed in the Services folder in your Library folder.
  4. Repeat the previous two steps for each of the remaining service workflow files.
