Publishing Device Information

A task common to all device management scenarios is to gather and share specific information about the devices. Quite often, a spreadsheet is the perfect container for holding and sharing this data.

The New Properties Table for Devices action can quickly query devices, gather data, and construct a new formatted table in a Numbers document.

(⬇ see below ) This workflow, constructed of two actions, will create a new table in Numbers, containing data for the connected devices.

Once the workflow is run, will query for device data, and construct a new table in Numbers  (⬇ see below ) 

NOTE: not all properties are available for use by the action, and may be disabled in the action view.
