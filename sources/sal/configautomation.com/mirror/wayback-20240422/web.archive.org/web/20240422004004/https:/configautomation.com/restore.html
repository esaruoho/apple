The “Restore Devices” action

The Restore Devices action clears all data on the connected devices and installs the latest version of iOS on them.

Input:ECID Identifiers for the connected devices to restore
Output:ECID Identifiers for the connected devices that were restored
Parameters:

This action has no user-settable parameters

The Action View

NOTE: Since the use of this action may include the downloading an archive file of the latest iOS version from Apple’s servers, an active broadband connection is required.
