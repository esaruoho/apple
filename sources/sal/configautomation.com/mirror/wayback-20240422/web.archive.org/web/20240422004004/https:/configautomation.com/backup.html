The “Backup Specified Devices” action

The Backup Specified Devices action creates backup archives for each of the connected devices whose ECID is passed as input to this action.

Input:ECID Identifiers for the connected devices to backup
Output:ECID Identifiers for the connected devices that were archived
Parameters:

This action has no user-settable parameters
