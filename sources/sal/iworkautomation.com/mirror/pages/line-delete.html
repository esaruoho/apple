Deleting Lines

After using the create grid of lines to develop a page layout design, you’ll want to remove the lines from the page. Here’s a script that will remove all of the lines on the current page in the frontmost Pages document:

Delete Lines from Current Page
  
01tell application "Pages"
02 activate
03 display dialog "Delete lines from the current page?" buttons {"Cancel", "Delete All", "Delete Unlocked"} default button 1
04 set deletionOption to the button returned of the result
05 tell document 1
06 tell current page
07 if deletionOption is "Delete All" then
08 set locked of every line to false
09 delete every line
10 else
11 delete (every line whose locked is false)
12 end if
13 end tell
14 end tell
15end tell
