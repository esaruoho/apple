Movies

Adding movies to a document can do much to increase reader comprehension by presenting concepts and ideas in a relatable format. For example, including a video of Earth from the International Space Station could be an excellent lead-in to an overview global financial priorities and realties.

The iWork Suite movie Class

movie n  [inh. iWork item ] : An audio clip

elements

contained by slides, document.

properties

file name ( text r/o) : The name of the movie file.

movie volume ( integer ) : The volume setting for the movie, from 0 (none) to 100 (full volume).

opacity ( integer ) : The opacity of the object, in percent.

reflection showing ( boolean ) : Is the movie displaying a reflection?

reflection value ( integer ) : The percentage of reflection of the movie, from 0 (none) to 100 (full).

repetition method ( none/‌loop/‌loop back and forth ) : If or how the audio clip repeats.

rotation ( integer ) : The rotation of the iWork item, in degrees from 0 to 359.

responds to

delete, exists

NOTE: In the current version of Pages the movie class does not support the use of the make command. See the script example below for details on how to import a movie file.

Import a Movie
  
01property defaultMovieWidth : 360
02 
03tell application "Pages"
04 activate
05 
06 try
07 -- PROMPT USER TO PICK MOVIE FILE
08 set movieFileHFSAlias to ¬
09 (choose file of type "public.movie" with prompt ¬
10 "Select the movie file to import:" default location (path to movies folder))
11 
12 -- CREATE DOCUMENT IF ONE DOES NOT EXIST
13 if not (exists document 1) then
14 make new document with properties ¬
15 {document template:template "Blank"}
16 end if
17 
18 tell document 1
19 -- IMPORT MOVIE FILE TO CURRENT PAGE
20 tell the current page
21 -- Pages does not support using the movie class with the make verb
22 -- Import the movie file by using the image class as the direct parameter
23 -- The returned object reference will be to the created movie item
24 set thisMovie to make new image with properties {file:movieFileHFSAlias}
25 --> movie 1 of sheet 1 of document "Document Name"
26 
27 -- ADJUST MOVIE PROPERTIES
28 tell thisMovie
29 -- adjust the size and position of the movie item...
30 -- using properties inherited from iWork Item class
31 set width to defaultMovieWidth
32 set position to {0, 0}
33 -- movie class properties
34 set reflection showing to true
35 set reflection value to 75
36 set movie volume to 90
37 set repetition method to none
38 -- place the locked property at the end because
39 -- you can't change the properties of a locked iWork item
40 set locked to true
41 end tell
42 end tell
43 end tell
44 on error errorMessage number errorNumber
45 if errorNumber is not -128 then
46 display alert ("ERROR " & errorNumber as string) message errorMessage
47 end if
48 end try
49end tell

Protected Video Files

iWork applications do not allow the import of copy-protected media files. Attempting to import such items may cause an error.
