Sheet

A sheet in a Numbers document, can be thought of as the equivalent of a page in a layout application’s document. A sheet can contain tables, images, videos, or any of the other data types supported by Numbers.

The Numbers 2009 Compatibility Suite supports tables as the only scriptable element of a sheet. Note that the sheet class has a single property “name”, whose value is the title of the sheet.

sheet n : A sheet in a document

elements

contains tables; contained by documents.

properties

name ( text ) : The sheet’s name.

responds to

delete, exists, make
