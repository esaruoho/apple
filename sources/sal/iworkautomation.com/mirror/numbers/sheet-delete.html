Deleting Sheets

The process of deleting sheets is accomplished using the delete command from the Standard Suite:

delete v  : Delete an object.

delete specifier : The object(s) to delete.

The delete command takes as its direct parameter, an object reference to the sheet or sheets to be removed from the Numbers document.

Delete Every Sheet Without Tables
  
01tell application "Numbers"
02 activate
03 tell the front document
04 repeat with i from (count of sheets) to 1 by -1
05 if the (count of tables of sheet i) is 0 then
06 delete sheet i
07 end if
08 end repeat
09 end tell
10end tell
