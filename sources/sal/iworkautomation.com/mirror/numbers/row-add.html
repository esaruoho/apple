Adding Rows

Adding rows to a table is accomplished using the add row above and add row below commands, which take as a direct parameter, a range of the table.

add row above v  : Add a row to the table above a specified range of cells.

add column after range

add row below v  : Add a row to the table below a specified range of cells.

add column before range

The script example below, adds a row, ending with a table summation cell, to the end of the currently selected table:

 (⬆ see above )  A table to which will be added a row, using the following script:

Add Table Summation Cell
  
01-- This script adds a table summation cell to the bottom right of the selected table
02-- NOTE: This script assumes the use of one row header and one column header
03tell application "Numbers"
04 activate
05 try
06 if not (exists document 1) then error number 1000
07 tell document 1
08 try
09 tell active sheet
10 set the selectedTable to ¬
11 (the first table whose class of selection range is range)
12 end tell
13 on error
14 error number 1001
15 end try
16 tell selectedTable
17 -- store the address of the current non-header cell area
18 set tableRangeAddress to (name of cell 2 of row 2) & ":" & (name of last cell)
19 -- add a row to the end of the table
20 add row below last row
21 tell last row
22 -- set style properties of new row
23 set background color to {0, 0, 0}
24 set text color to {65535, 65535, 65535}
25 -- derive the range address of the cells to merge
26 set the mergeRangeAddress to (name of second cell) & ":" & (name of cell -2)
27 end tell
28 -- merge all cells of the new row except the frist and last ones
29 merge range mergeRangeAddress
30 tell last row
31 -- set the contents and style of merged cells
32 set alignment of second cell to right
33 set value of second cell to "TOTAL"
34 end tell
35 -- set formula of last cell to be SUM of table
36 set the value of the last cell to ("=SUM(" & tableRangeAddress & ")")
37 end tell
38 end tell
39 on error errorMessage number errorNumber
40 if errorNumber is 1000 then
41 set alertString to "MISSING RESOURCE"
42 set errorMessage to "Please create or open a document before running this script."
43 else if errorNumber is 1001 then
44 set alertString to "SELECTION ERROR"
45 set errorMessage to "Please select a table before running this script."
46 else
47 set alertString to "EXECUTION ERROR"
48 end if
49 display alert alertString message errorMessage buttons {"Cancel"}
50 error number -128
51 end try
52end tell
