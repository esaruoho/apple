Cell Data Formats

Cells have a format property, inherited from the range class, that determines how the cell’s data is parsed and displayed. By default, a new cell’s format is set to automatic, and will attempt to interpret the inserted data into the correct format. Set the format property if the data, such as a numeric value, can be interpreted to be used by more than one format, otherwise leave the format property set to the default of automatic.

The following script will create a small table displaying the various available cell formats:

Create Cell Format Table
  
01tell application "Numbers"
02 activate
03 if not (exists document 1) then error number -128
04 tell document 1
05 tell active sheet
06 -- make a new table
07 set thisTable to make new table with properties ¬
08 {row count:16, column count:2, name:"Cell Formats"}
09 tell thisTable
10 -- a list of the supported formats
11 set theseFormats to {automatic, checkbox, currency, date and time, fraction, number, percent, pop up menu, scientific, slider, stepper, text, duration, rating, numeral system}
12 -- placeholder data
13 set formatData to {"", true, 1234.02, (get current date), 0.1, 123.89, 0.75, "", 64, 64, 35, "HOWDY", 4.5, 3, 4569.123}
14 -- set the values of the first column with the names of the formats
15 tell column 1
16 repeat with i from 1 to the count of theseFormats
17 set the value of cell (i + 1) to (item i of theseFormats) as string
18 end repeat
19 end tell
20 -- set the formats of the column cells, and add placeholder data
21 tell column 2
22 repeat with i from 1 to the count of theseFormats
23 set the format of cell (i + 1) to (item i of theseFormats)
24 set the value of cell (i + 1) to (item i of formatData)
25 end repeat
26 end tell
27 end tell
28 end tell
29 end tell
30end tell
