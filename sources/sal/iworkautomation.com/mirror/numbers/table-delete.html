Deleting Tables

The table class can be the target of the delete command.

delete v  : Delete an object.

delete specifier : The object(s) to delete.

The good use of the delete command is in conjunction with a whose clause to locate tables by examining the value of their properties, such as name.

Delete Old Tables
  
01tell application "Numbers"
02 activate
03 if not (exists document 1) then error number -128
04 
05 tell the front document
06 delete (every table of every sheet whose name ends with "2012")
07 end tell
08end tell
