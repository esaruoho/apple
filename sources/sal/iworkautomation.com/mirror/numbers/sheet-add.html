Adding Sheets

Sheets are added to a document by using the make command of the Standard Suite.

make v  : Create a new object.

make

new class : The class of the new object.

[ at location specifier ] : The location at which to insert the object.

[ with data any ] : The initial contents of the object.

[ with properties record ] : The initial values for properties of the object.

⇒ specifier : The new object.

When creating a sheet, the optional with properties parameter of the make command is supported, so that sheets can be created and named with a single statement.

New Document with Sheet for Every Month
  
01set the monthNames to {"January", "February", "March", "April", "May", "June", "July", "August", "September", "October", "November", "December"}
02set thisYearName to (year of (current date)) as string
03 
04tell application "Numbers"
05 activate
06 -- confirmation dialog
07 display dialog "This script will create a new document containing a sheet, " & ¬
08 "with a named table, for every month of the year." buttons ¬
09 {"Cancel", "Continue"} default button 2 with icon 1
10 -- make a new document
11 set thisDocument to make new document
12 -- create a sheet with a named table for every month
13 tell thisDocument
14 repeat with i from 1 to the count of the monthNames
15 set thisMonthName to item i of the monthNames
16 set thisTableName to thisMonthName & "-" & thisYearName
17 if i is 1 then
18 -- name the sheet
19 set the name of sheet 1 to thisMonthName
20 tell sheet 1
21 -- delete default table and create new one
22 delete every table
23 make new table with properties {name:thisTableName}
24 end tell
25 else
26 -- make a new sheet
27 set thisSheet to ¬
28 make new sheet with properties {name:thisMonthName}
29 tell thisSheet
30 -- delete default table and create new one
31 delete every table
32 make new table with properties {name:thisTableName}
33 end tell
34 end if
35 end repeat
36 delay 0.5
37 -- display the first sheet by selecting table on current sheet, then table on first sheet
38 set selection range of table 1 of sheet 12 to cell range of table 1 of sheet 12
39 delay 1
40 set selection range of table 1 of sheet 1 to cell range of table 1 of sheet 1
41 end tell
42 display dialog "Document created." buttons {"OK"} default button 1 with icon 1
43end tell

The script below, adds a sheet, titled for every day of the current month, to the front document.

New Document with Sheet for Every Day of This Month
  
01copy my deriveNameIndexLengthYearOfCurrentMonth() to ¬
02 {thisMonthName, thisMonthIndex, monthLengthInDays, thisYearName}
03 
04tell application "Numbers"
05 activate
06 -- confirmation dialog
07 display dialog "This script will create a new document " & ¬
08 "containing a sheet for every day of the month." buttons ¬
09 {"Cancel", "Continue"} default button 2 with icon 1
10 -- make a new document
11 set thisDocument to make new document
12 -- create a sheet with a named table for every month
13 tell thisDocument
14 repeat with i from 1 to monthLengthInDays
15 set the sheetName to ¬
16 (thisMonthName & space & i & space & thisYearName) as string
17 set the tableName to ¬
18 (thisMonthIndex & "-" & i & "-" & thisYearName) as string
19 if i is 1 then
20 -- name the sheet
21 set the name of sheet 1 to the sheetName
22 -- delete default table and create new one
23 tell sheet 1
24 delete every table
25 make new table with properties {name:tableName}
26 end tell
27 else
28 -- make a new sheet
29 set thisSheet to ¬
30 make new sheet with properties {name:sheetName}
31 tell thisSheet
32 -- delete default table and create new one
33 delete every table
34 make new table with properties {name:tableName}
35 end tell
36 end if
37 end repeat
38 delay 0.5
39 -- display the first sheet by selecting table on current sheet, then table on first sheet
40 set selection range of table 1 of sheet 12 to cell range of table 1 of sheet 12
41 delay 1
42 set selection range of table 1 of sheet 1 to cell range of table 1 of sheet 1
43 end tell
44 display dialog "Document created." buttons {"OK"} default button 1 with icon 1
45end tell
46 
47on deriveNameIndexLengthYearOfCurrentMonth()
48 set thisDate to (current date)
49 set day of thisDate to 1
50 set thisMonth to month of thisDate
51 set thisMonthName to thisMonth as string
52 set thisMonthIndex to thisMonth as integer
53 set thisYearName to (year of thisDate) as string
54 repeat with i from 1 to 32
55 set thisDate to thisDate + (1 * days)
56 if month of thisDate is not thisMonth then
57 set monthLengthInDays to i
58 exit repeat
59 end if
60 end repeat
61 return {thisMonthName, thisMonthIndex, monthLengthInDays, thisYearName}
62end deriveNameIndexLengthYearOfCurrentMonth
