Text Item Styling

The iWork Text Suite provides the classes and properties for manipulating text in all iWork applications. Using this suite a script can set the font (typeface), size, and color of the object text placed within text items and shapes, as demonstrated in this video of a presentation created by the example script from this page:

Here is the dictionary listing for the iWork Text Suite:

The iWork Text Suite

rich text n   pl  rich text : This provides the base rich text class for all iWork applications.

elements

contains characters, paragraphs, words

properties

color ( RGB color | text ) : The color of the font. Expressed as an RGB value consisting of a list of three color values from 0 to 65535. ex: Blue = {0, 0, 65535}. In addition, the lowercase names of the following standard colors can be used in place of RGB color value lists: "black", "blue", "brown", "cyan", "green", "magenta", "orange", "purple", "red", "yellow", and "white".

font ( text ) : The name of the font. Can be the PostScript name, such as: “TimesNewRomanPS-ItalicMT”, or display name: “Times New Roman Italic”. TIP: Use the Font Book application get the information about a typeface.

size ( integer ) : The size of the font (in points).

 

character n  [inh. rich text ] : One of some text’s characters.

elements

contained by rich text, paragraphs, words.

 

paragraph n  [inh. rich text ] : One of some text’s paragraphs.

elements

contained by rich text.

 

word n  [inh. rich text ] : One of some text’s words.

elements

contained by rich text, paragraphs.

The following script demonstrates how to apply styling to a text item’s object text:

Styling Text
  
01tell application "Keynote"
02 activate
03 
04 set thisDocument to ¬
05 make new document with properties ¬
06 {document theme:theme "Black", width:1920, height:1080}
07 
08 tell thisDocument
09 set documentHeight to its height
10 
11 set the base slide of the first slide to master slide "Blank"
12 
13 tell slide 1
14 -- create a new text item
15 set thisDisplayText to "Abracadabra"
16 set thisTextItem to ¬
17 make new text item with properties {object text:thisDisplayText}
18 tell thisTextItem
19 -- set type size
20 set the size of its object text to 144
21 -- set typeface
22 set the font of its object text to "Zapfino"
23 -- adjust its vertical positon
24 copy its position to {currentHorizontal, currentVertical}
25 set its position to {currentHorizontal, documentHeight div 3}
26 end tell
27 end tell
28 
29 duplicate slide 1 to after slide 1
30 
31 tell slide 2
32 set thisTextItem to its first text item
33 tell thisTextItem
34 -- set color
35 repeat with i from 1 to the length of thisDisplayText
36 set thisRGBColorValue to my generateRandomRGBColorValue()
37 set the color of character i of object text to thisRGBColorValue
38 end repeat
39 end tell
40 end tell
41 
42 set the transition properties of the first slide to ¬
43 {transition effect:sparkle, transition duration:2.0 ¬
44 , transition delay:1.0, automatic transition:true}
45 end tell
46 
47 start thisDocument from first slide of thisDocument
48end tell
49 
50on generateRandomRGBColorValue()
51 set RedValue to random number from 0 to 65535
52 set GreenValue to random number from 0 to 65535
53 set BlueValue to random number from 0 to 65535
54 return {RedValue, GreenValue, BlueValue}
55end generateRandomRGBColorValue
