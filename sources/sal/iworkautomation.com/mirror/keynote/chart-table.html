Table & Charts from Shared Data

A compelling reason for using automation is its ability to quickly display data in a variety of formats. For example, a single data set can be displayed as a table, or as a chart, or as both — in seconds. With the AppleScript support in Keynote, connecting data to display becomes a hands-free process, as demonstrated by the following script that generates a presentation containing slides showing a table and multiple charts from a single data set:  (⬇ see below ) 

Table & Charts From Shared Data
  
01tell application "Keynote"
02 
03 -- STOP ANY PLAYING PRESENTATION
04 if playing is true then stop the front document
05 
06 activate
07 
08 -- MAKE NEW DOCUMENT
09 set thisDocument to make new document with properties ¬
10 {height:764, width:1024, document theme:theme "Black"}
11 
12 tell thisDocument
13 
14 -- THE SOURCE DATA
15 set rowNames to {"ROW A", "ROW B", "ROW C"}
16 set columnNames to {"COL 1", "COL 2", "COL 3", "COL 4"}
17 set rowCount to count of rowNames
18 set columnCount to count of columnNames
19 set thisData to my generateRandomData(rowCount, columnCount)
20 
21 -- THE EXISTING SLIDE
22 tell slide 1
23 set the base slide to master slide "Title - Top" of thisDocument
24 set the object text of the default title item to "Table"
25 
26 -- MAKE A TABLE FROM DATA
27 -- include headers in counts
28 set the rowCount to (the count of thisData) + 1
29 set the columnCount to (the count of (item 1 of thisData)) + 1
30 
31 -- make the table
32 set thisTable to ¬
33 make new table with properties ¬
34 {row count:rowCount, column count:columnCount}
35 
36 -- populate the table
37 tell thisTable
38 -- column headers
39 repeat with i from 1 to the the count of the columnNames
40 set the value of cell (i + 1) of row 1 to item i of columnNames
41 end repeat
42 
43 -- row headers
44 repeat with i from 1 to the the count of the rowNames
45 set the value of cell (i + 1) of column 1 to item i of rowNames
46 end repeat
47 
48 -- insert data, which is assumed to be grouped by rows
49 repeat with q from 1 to the count of thisData
50 set thisDataItem to item q of thisData
51 repeat with i from 1 to the the count of thisDataItem
52 set the value of cell (i + 1) of row (q + 1) to item i of thisDataItem
53 end repeat
54 end repeat
55 end tell
56 end tell
57 
58 -- MAKE NEW SLIDE
59 set thisSlide to make new slide with properties {base slide:master slide "Title - Top"}
60 tell thisSlide
61 set the object text of the default title item to "Grouped by Column"
62 
63 -- MAKE A CHART FROM DATA (GROUPED BY COLUMN)
64 add chart row names rowNames column names columnNames ¬
65 data thisData type line_2d group by chart column
66 end tell
67 
68 -- MAKE NEW SLIDE
69 set thisSlide to make new slide with properties {base slide:master slide "Title - Top"}
70 tell thisSlide
71 set the object text of the default title item to "Grouped by Row"
72 
73 -- MAKE A CHART FROM DATA (GROUPED BY ROW)
74 add chart row names rowNames column names columnNames ¬
75 data thisData type line_2d group by chart row
76 end tell
77 end tell
78 
79 -- PLAY THE PRESENTATION
80 start thisDocument from first slide of thisDocument
81 
82end tell
83 
84-- SUB-ROUTINE FOR GENERATING RANDOM ROW/COLUMN DATA
85on generateRandomData(rowCount, columnCount)
86 set thisData to {}
87 repeat rowCount times
88 set thisDataItem to {}
89 repeat columnCount times
90 set thisValue to random number from 0 to 100
91 set the end of thisDataItem to thisValue
92 end repeat
93 set the end of thisData to thisDataItem
94 end repeat
95 return thisData
96end generateRandomData
