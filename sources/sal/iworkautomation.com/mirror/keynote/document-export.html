Exporting Documents

Exporting Keynote documents is done using the export command from the Keynote Suite.

export v  : Export a presentation to another file(s).

export document : The presentation to export.

to (file ) : A reference to the file (folder) to be created. When exporting images or HTML, a new folder, containing the exported items, is created instead of a file.

as ( HTML / ‌QuickTime movie /‌ PDF /‌ slide images / ‌Microsoft PowerPoint /‌ Keynote 09 ) : The format in which the document is to be exported.

[ with properties (export options ) ] : An AppleScript record containing property:value pairings for the initial values for properties of the exported object.

export options n : The export options for the Keynote export command.

properties

all stages ( boolean ) : Include each stage of the builds.

borders ( boolean ) : Add borders around the slides.

compression factor ( real ) : Compressed image quality, ranging from 0.0 (maximum compression, lowest quality) to 1.0 (lossless compression, highest quality). This option only pertains to JPEG format images.

date ( boolean ) : Include the current date.

export style ( IndividualSlides /‌ SlideWithNotes /‌ Outline /‌ Handouts ) : Indicate whether to include notes, etc. (applies only to printing)

image format ( JPEG /‌ PNG /‌ TIFF ) : The format to use for the resulting images.

movie format ( small /‌ medium /‌ large ) : The desired size for exported movie: 360P, 540P, 720P

password ( text ) : The password to use when the exported PDF is encrypted.

password hint ( text ) : The password hint.

PDF image quality ( good/better/best ) : (Keynote 6.6) The quality of the images in the exported PDF file.

rawKPF ( boolean ) : Should the document be exported in raw KPF?

skipped slides ( boolean ) : Should the exported document include skipped slides?

slide numbers ( boolean ) : Should the exported document include slide numbers?

NOTE: The Keynote version 6.6 scripting dictionary includes options for indicating the quality of images used when the presentation is exported as a PDF file.

Shortcuts to the script examples:

Export to Images, Export to Movie, Export to HTML, Export to Microsoft PowerPoint, Export to PDF

Export to Images

Here’s an example script that exports the front document to image files.

Export Slides to Images
  
01property exportFormat : "PNG" -- "TIFF" "JPEG" "PNG"
02property includeSkippedSlides : false
03property compressionFactor : 1.0
04 
05-- THE DESTINATION FOLDER
06-- (see the "path" to command in the Standard Additions dictionary for other locations, such as documents folder, movies folder, desktop folder)
07set the defaultDestinationFolder to (path to pictures folder)
08 
09tell application "Keynote"
10 activate
11 try
12 if playing is true then tell the front document to stop
13 
14 if not (exists document 1) then error number -128
15 
16 -- DERIVE NAME FOR NEW FOLDER FROM NAME OF THE FRONT DOCUMENT
17 set documentName to the name of the front document
18 if documentName ends with ".key" then ¬
19 set documentName to text 1 thru -5 of documentName
20 
21 -- CREATE AN EXPORT DESTINATION FOLDER
22 -- IMPORTANT: IT’S ADVISED TO ALWAYS CREATE a NEW DESTINATION FOLDER, AS THE CONTENTS OF ANY TARGETED FOLDER WILL BE OVERWRITTEN
23 tell application "Finder"
24 set newFolderName to documentName
25 set incrementIndex to 1
26 repeat until not (exists folder newFolderName of defaultDestinationFolder)
27 set newFolderName to documentName & "-" & (incrementIndex as string)
28 set incrementIndex to incrementIndex + 1
29 end repeat
30 set the targetFolder to ¬
31 make new folder at defaultDestinationFolder with properties ¬
32 {name:newFolderName}
33 set the targetFolderHFSPath to targetFolder as string
34 end tell
35 
36 -- EXPORT THE DOCUMENT
37 if exportFormat is "PNG" then
38 -- EXPORT THE FRONT DOCUMENT TO PNG IMAGES
39 export front document as slide images to file targetFolderHFSPath with properties ¬
40 {image format:PNG, skipped slides:includeSkippedSlides}
41 else if exportFormat is "JPEG" then
42 -- EXPORT THE FRONT DOCUMENT TO JPEG IMAGES
43 export front document as slide images to file targetFolderHFSPath with properties ¬
44 {image format:JPEG, skipped slides:includeSkippedSlides ¬
45 , compression factor:compressionFactor}
46 else if exportFormat is "TIFF" then
47 -- EXPORT THE FRONT DOCUMENT TO TIFF IMAGES
48 export front document as slide images to file targetFolderHFSPath with properties ¬
49 {image format:TIFF, skipped slides:includeSkippedSlides ¬
50 , compression factor:compressionFactor}
51 end if
52 on error errorMessage number errorNumber
53 display alert "EXPORT PROBLEM" message errorMessage
54 error number -128
55 end try
56end tell
57 
58-- OPEN THE DESTINATION FOLDER
59tell application "Finder"
60 activate
61 open the targetFolder
62end tell

TIP: To see how to convert the previous script to a droplet, check out the Movie Export Droplet example on this website.

Export to Movie

Here’s a script for exporting the Keynote presentation to a movie file:

Export Front Document to Movie
  
01property movieFileExtension : "m4v"
02 
03-- THE DESTINATION FOLDER
04-- (see the "path" to command in the Standard Additions dictionary for other locations, such as documents folder, pictures folder, desktop folder)
05set the defaultDestinationFolder to (path to movies folder)
06 
07tell application "Keynote"
08 activate
09 try
10 if playing is true then tell the front document to stop
11 
12 if not (exists document 1) then error number -128
13 
14 -- DERIVE NAME AND FILE PATH FOR NEW MOVIE FILE
15 set documentName to the name of the front document
16 if documentName ends with ".key" then ¬
17 set documentName to text 1 thru -5 of documentName
18 
19 tell application "Finder"
20 set newMovieFileName to documentName & "." & movieFileExtension
21 set incrementIndex to 1
22 repeat until not (exists document file newMovieFileName of defaultDestinationFolder)
23 set newMovieFileName to ¬
24 documentName & "-" & (incrementIndex as string) & "." & movieFileExtension
25 set incrementIndex to incrementIndex + 1
26 end repeat
27 end tell
28 set the targetFileHFSpath to (defaultDestinationFolder as string) & newMovieFileName
29 
30 -- EXPORT THE DOCUMENT
31 with timeout of 1200 seconds
32 export front document to file targetFileHFSPath ¬
33 as QuickTime movie with properties {movie format:large}
34 end timeout
35 
36 on error errorMessage number errorNumber
37 display alert "EXPORT PROBLEM" message errorMessage
38 error number -128
39 end try
40end tell
41 
42-- SHOW THE NEW MOVIE FILE
43tell application "Finder"
44 activate
45 reveal document file targetFileHFSPath
46end tell

Export to Microsoft PowerPoint

This version of the export script will export the front Keynote presentation to a Microsoft PowerPoint file. Presentations containing media, such as movies and/or audio files, must be exported to the newer file format (PPTX), while presentations without media can be exported to the Microsoft PowerPoint 1997-2003 compatible format (PPT).

Export to Microsoft PowerPoint
  
01-- Export the presentation as a MSPowerPoint 1997-2003 compatible file?
02property attemptCompatibleExport : true
03 
04-- THE DESTINATION FOLDER
05-- (see the "path to" command in the Standard Additions dictionary for other locations, such as movies folder, pictures folder, desktop folder)
06set the defaultDestinationFolder to (path to documents folder)
07 
08tell application "Keynote"
09 activate
10 try
11 if playing is true then tell the front document to stop
12 
13 if not (exists document 1) then error number -128
14 
15 tell the front document
16 set documentName to its name
17 if documentName ends with ".key" then ¬
18 set documentName to text 1 thru -5 of documentName
19 set movieCount to the count of every movie of every slide
20 set audioClipCount to the count of every audio clip of every slide
21 end tell
22 
23 -- Check for media. Presentations containing media must be saved as PPTX
24 if movieCount is 0 and audioClipCount is 0 and attemptCompatibleExport is true then
25 set MicrosoftPowerPointFileExtension to "ppt"
26 else
27 set MicrosoftPowerPointFileExtension to "pptx"
28 end if
29 
30 tell application "Finder"
31 set newExportItemName to documentName & "." & MicrosoftPowerPointFileExtension
32 set incrementIndex to 1
33 repeat until not (exists document file newExportItemName of defaultDestinationFolder)
34 set newExportItemName to ¬
35 documentName & "-" & (incrementIndex as string) & "." & MicrosoftPowerPointFileExtension
36 set incrementIndex to incrementIndex + 1
37 end repeat
38 end tell
39 set the targetFileHFSpath to (defaultDestinationFolder as string) & newExportItemName
40 
41 -- EXPORT THE DOCUMENT
42 with timeout of 1200 seconds
43 export front document to file targetFileHFSPath as Microsoft PowerPoint
44 end timeout
45 
46 set exportedItempath to targetFileHFSPath
47 
48 on error errorMessage number errorNumber
49 display alert "EXPORT PROBLEM" message errorMessage
50 error number -128
51 end try
52end tell
53 
54-- SHOW THE NEW EXPORTED FILE
55tell application "Finder"
56 activate
57 set exportedFile to (document file exportedItemPath)
58 set extension hidden of exportedFile to false
59 reveal exportedFile
60end tell

Export to PDF

This script exports the frontmost Keynote presentation to a PDF file. Set the boolean value of the useEncryptionDefaultValue property to either true or false to determine whether you will be prompted for the password to use in encrypting the PDF file.

Export to PDF
  
01property exportFileExtension : "pdf"
02property useEncryptionDefaultValue : true
03 
04-- THE DESTINATION FOLDER
05-- (see the "path" to command in the Standard Additions dictionary for other locations, such as movies folder, pictures folder, desktop folder)
06set the defaultDestinationFolder to (path to documents folder)
07 
08set usePDFEncryption to useEncryptionDefaultValue
09tell application "Keynote"
10 activate
11 try
12 if playing is true then tell the front document to stop
13 
14 if not (exists document 1) then error number -128
15 
16 if usePDFEncryption is true then
17 -- PROMPT FOR PASSWORD (OPTIONAL)
18 repeat
19 display dialog "Enter a password for the PDF file:" default answer ¬
20 "" buttons {"Cancel", "No Password", "OK"} ¬
21 default button 3 with hidden answer
22 copy the result to ¬
23 {button returned:buttonPressed, text returned:firstPassword}
24 if buttonPressed is "No Password" then
25 set usePDFEncryption to false
26 exit repeat
27 else
28 display dialog "Enter the password again:" default answer ¬
29 "" buttons {"Cancel", "No Password", "OK"} ¬
30 default button 3 with hidden answer
31 copy the result to ¬
32 {button returned:buttonPressed, text returned:secondPassword}
33 if buttonPressed is "No Password" then
34 set usePDFEncryption to false
35 exit repeat
36 else
37 if firstPassword is not secondPassword then
38 display dialog "Passwords do no match." buttons ¬
39 {"Cancel", "Try Again"} default button 2
40 else
41 set providedPassword to the firstPassword
42 set usePDFEncryption to true
43 exit repeat
44 end if
45 end if
46 end if
47 end repeat
48 end if
49 
50 -- DERIVE NAME AND FILE PATH FOR NEW EXPORT FILE
51 set documentName to the name of the front document
52 if documentName ends with ".key" then ¬
53 set documentName to text 1 thru -5 of documentName
54 
55 tell application "Finder"
56 set exportItemFileName to documentName & "." & exportFileExtension
57 set incrementIndex to 1
58 repeat until not (exists document file exportItemFileName of defaultDestinationFolder)
59 set exportItemFileName to ¬
60 documentName & "-" & (incrementIndex as string) & "." & exportFileExtension
61 set incrementIndex to incrementIndex + 1
62 end repeat
63 end tell
64 set the targetFileHFSpath to (defaultDestinationFolder as string) & exportItemFileName
65 
66 -- EXPORT THE DOCUMENT
67 with timeout of 1200 seconds
68 if usePDFEncryption is true then
69 export front document to file targetFileHFSpath ¬
70 as PDF with properties {password:providedPassword}
71 else
72 export front document to file targetFileHFSpath as PDF
73 end if
74 end timeout
75 
76 on error errorMessage number errorNumber
77 if errorNumber is not -128 then
78 display alert "EXPORT PROBLEM" message errorMessage
79 end if
80 error number -128
81 end try
82end tell
83 
84-- SHOW THE NEW PDF FILE
85tell application "Finder"
86 activate
87 reveal document file targetFileHFSpath
88end tell
