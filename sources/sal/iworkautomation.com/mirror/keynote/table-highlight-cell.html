Highlight Cell with Transition

“And as you can see, the third quarter results for W-21 are increasing…” blah, blah, blah. (insert sounds like Charlie Brown’s teacher).

PRESENTATION RULE #5: Don’t assume that your audience sees what you see, or understands what you’re saying. Make it easy for them to recognize and absorb the point you’re making.

This is especially true when presenting tables. If there’s a cell value to highlight, then highlight it using the following script. Ah, peace and understanding are attained.

DO THIS ►DOWNLOAD the example presentation.

(⬆ see above ) A cell selected in a table on a slide. (⬇ see below ) A duplicate slide with the table formated to highlight the cell selected in the source table. Note that the row and column headers for the targeted cell are lighter, while the other table values are shaded darker.

Here’s what the effect looks like in a presentation.
(Click Movie to Play)

This script will duplicate the slide containing the selected table cell, and format the duplicated table to highlight the previously selected cell. In addition, the script will apply a transition to the source slide, and for your review, will start the presentation from the source slide. Enjoy!

IMPORTANT TIP: Learn how to save and run your favorite scripts using the system-wide Script Menu.

Highlight Selected Cell with Transition
  
01property selectedRangeBackgroundColor : {65535, 65535, 65535}
02property selectedRangeTextColor : {0, 0, 0}
03property backgroundDataColor : {39321, 39321, 39321}
04property headerHightlightTextColor : {65535, 65535, 65535}
05property lightnessValue : 15000
06 
07tell application "Keynote"
08 activate
09 try
10 if not (exists document 1) then error number 1000
11 tell document 1
12 set sourceSlide to the current slide
13 tell sourceSlide
14 set thisSlideNumber to its slide number
15 try
16 set the selectedTable to ¬
17 (the first table whose class of selection range is range)
18 on error
19 error number 1001
20 end try
21 tell selectedTable
22 set the selectionRangeName to the name of selection range
23 set the headerColumnCount to header column count
24 set the headerRowCount to header row count
25 set the footerRowCount to footer row count
26 set the tableDataRangeStart to ¬
27 the name of cell (headerColumnCount + 1) of row (headerRowCount + 1)
28 set the tableDataRangeEnd to ¬
29 the name of cell -1 of row ((footerRowCount + 1) * -1)
30 set tableDataRange to range (tableDataRangeStart & ":" & tableDataRangeEnd)
31 end tell
32 end tell
33 duplicate sourceSlide to after sourceSlide
34 set duplicateSlide to slide (thisSlideNumber + 1)
35 tell duplicateSlide
36 set the transition properties to {automatic transition:false}
37 tell table 1
38 set the tableDataRangeStart to ¬
39 the name of cell (headerColumnCount + 1) of row (headerRowCount + 1)
40 set the tableDataRangeEnd to ¬
41 the name of cell -1 of row ((footerRowCount + 1) * -1)
42 set tableDataRange to range (tableDataRangeStart & ":" & tableDataRangeEnd)
43 set text color of tableDataRange to backgroundDataColor
44 
45 if headerColumnCount is not 0 then
46 copy background color of last cell of column 1 to {RValue, GValue, BValue}
47 set lighterColumnColor to my lightenRGB(RValue, GValue, BValue, lightnessValue)
48 tell columns 1 thru headerColumnCount
49 set text color to backgroundDataColor
50 end tell
51 end if
52 
53 if headerRowCount is not 0 then
54 copy background color of last cell of row 1 to {RValue, GValue, BValue}
55 set lighterRowColor to my lightenRGB(RValue, GValue, BValue, lightnessValue)
56 tell rows 1 thru headerRowCount
57 set text color to backgroundDataColor
58 end tell
59 end if
60 
61 tell range selectionRangeName
62 if headerColumnCount is not 0 then
63 set the text color of cells 1 thru ¬
64 headerColumnCount of its row to headerHightlightTextColor
65 set the background color of cells 1 thru ¬
66 headerColumnCount of its row to lighterColumnColor
67 end if
68 if headerRowCount is not 0 then
69 set the text color of cells 1 thru ¬
70 headerRowCount of its column to headerHightlightTextColor
71 set the background color of cells 1 thru ¬
72 headerRowCount of its column to lighterRowColor
73 end if
74 set background color to selectedRangeBackgroundColor
75 set text color to selectedRangeTextColor
76 end tell
77 end tell
78 end tell
79 tell sourceSlide
80 set the transition properties to ¬
81 {transition effect:dissolve ¬
82 , transition delay:0 ¬
83 , transition duration:0.75 ¬
84 , automatic transition:false}
85 end tell
86 end tell
87 start document 1 from sourceSlide
88 on error errorMessage number errorNumber
89 if errorNumber is 1000 then
90 set alertString to "MISSING RESOURCE"
91 set errorMessage to "Please create or open a document before running this script."
92 else if errorNumber is 1001 then
93 set alertString to "SELECTION ERROR"
94 set errorMessage to "Please select a table before running this script."
95 else
96 set alertString to "EXECUTION ERROR"
97 end if
98 if errorNumber is not -128 then
99 display alert alertString message errorMessage buttons {"Cancel"}
100 end if
101 error number -128
102 end try
103end tell
104 
105on lightenRGB(RValue, GValue, BValue, lightenAmount)
106 if RValue + lightenAmount is greater than 65535 then
107 set RValue to 65535
108 else
109 set RValue to RValue + lightenAmount
110 end if
111 if GValue + lightenAmount is greater than 65535 then
112 set GValue to 65535
113 else
114 set GValue to GValue + lightenAmount
115 end if
116 if BValue + lightenAmount is greater than 65535 then
117 set BValue to 65535
118 else
119 set BValue to BValue + lightenAmount
120 end if
121 return {RValue, GValue, BValue}
122end lightenRGB
