Deleting Slides

The delete command from the Standard Suite can be used to remove slides from a document.

Here are some example uses of the delete command:

Delete All Skipped Slides
  
01tell application "Keynote"
02 activate
03 if not (exists front document) then error number -128
04 display dialog "This script will delete every skipped slide of the frontmost presentation." & ¬
05 return & return & ¬
06 "Continue?" buttons {"Cancel", "Delete Slides"} default button 1 with icon 2
07 tell the front document
08 delete (every slide whose skipped is true)
09 end tell
10end tell

 

Delete All Slides Whose Title Contains “2012”
  
01property matchString : "2012"
02 
03tell application "Keynote"
04 activate
05 if not (exists front document) then error number -128
06 display dialog "This script will delete slides from the frontmost presentation." & ¬
07 return & return & ¬
08 "Continue?" buttons {"Cancel", "Delete Slides"} default button 1 with icon 2
09 tell the front document
10 delete (every slide where the object text of its default title item contains matchString)
11 end tell
12end tell
