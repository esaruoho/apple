Image Slides from Files

Here is a script that demonstrates the technique for creating new image slides from selected image files. You have the option to use the standard image import scaling or sizing the image to fill the slide.

(⬆ see above ) Image added to slide using the Keynote application’s default import methodology: if an image dimension is smaller than a corresponding slide dimension, then it is added at normal size; if an image dimension is greater than a corresponding slide dimension, then it is scaled to the corresponding slide dimension. The added image is centered in the window. In the example above, both dimensions of the image are smaller than their corresponding slide dimensions.

(⬇ see below ) Image added to slide using Scale to Fill method: adjust the image as needed (scale up or down) so that all the slide area is covered.

Image Slides from Files
  
01tell application "Keynote"
02 activate
03 if playing is true then tell the front document to stop
04 
05 set theseImageFiles to ¬
06 (choose file of type "public.image" with prompt ¬
07 "Select the image file(s) to import:" default location ¬
08 (path to pictures folder) with multiple selections allowed )
09 
10 display dialog "Scale the images using which method:" buttons ¬
11 {"Cancel", "Scale to Fill", "Default (Fit & Center)"} default button 3
12 if the button returned of the result is "Scale to Fill" then
13 set scaleToFill to true
14 else
15 set scaleToFill to false
16 end if
17 
18 repeat with i from 1 to the count of theseImageFiles
19 set thisImageFile to item i of theseImageFiles
20 if not (exists document 1) then
21 set thisDocument to make new document
22 set thisSlide to the first slide of thisDocument
23 set the base slide of thisSlide to master slide "Blank" of thisDocument
24 else
25 tell the front document
26 set thisSlide to make new slide with properties ¬
27 {base slide:master slide "Blank"}
28 end tell
29 end if
30 my addImageFileToSlide(thisSlide, thisImageFile, scaleToFill)
31 end repeat
32 
33end tell
34 
35on addImageFileToSlide(thisSlide, thisImageFile, scaleToFill)
36 tell application "Keynote"
37 try
38 activate
39 set queryResult to my getImageDimensions(POSIX path of thisImageFile)
40 if queryResult is false then
41 error number 10000
42 else
43 copy queryResult to {imageWidth, imageHeight}
44 end if
45 tell the front document
46 set documentWidth to its width
47 set documentHeight to its height
48 end tell
49 tell thisSlide
50 if scaleToFill is false then
51 set thisImage to make new image with properties {file:thisImageFile}
52 tell thisImage
53 set thisImageWidth to its width
54 set thisImageHeight to its height
55 -- center image
56 set position of it to ¬
57 {(documentWidth - thisImageWidth) / 2 ¬
58 , (documentHeight - thisImageHeight) / 2}
59 end tell
60 else -- scale to fill
61 -- figure new image height based on setting new image width to slide width
62 set newImageHeight to (imageHeight * documentWidth) / imageWidth
63 -- check to see if new image height is greater than or equal to document height
64 if newImageHeight is greater than or equal to documentHeight then
65 set newImageWidth to documentWidth
66 -- center image vertically
67 set verticalOffset to ((newImageHeight - documentHeight) / 2) * -1
68 set horizontalOffset to 0
69 else
70 -- scale image height to match slide height
71 set newImageHeight to documentHeight
72 set newImageWidth to (documentHeight * imageWidth) / imageHeight
73 -- center image horizontally
74 set verticalOffset to 0
75 set horizontalOffset to ((newImageWidth - documentWidth) / 2) * -1
76 end if
77 -- import, scale, and position image
78 set thisImage to make new image with properties ¬
79 {file:thisImageFile ¬
80 , width:newImageWidth ¬
81 , height:newImageHeight ¬
82 , position:{horizontalOffset, verticalOffset}}
83 end if
84 end tell
85 on error errorMessage number errorNumber
86 if errorNumber is not -128 then
87 display alert errorNumber message errorMessage
88 end if
89 error number -128
90 end try
91 end tell
92end addImageFileToSlide
93 
94on getImageDimensions(thisImageFilePOSIXPath)
95 try
96 set imageWidth to ¬
97 (do shell script "mdls -raw -name kMDItemPixelWidth " & ¬
98 quoted form of thisImageFilePOSIXPath) as integer
99 set imageHeight to ¬
100 (do shell script "mdls -raw -name kMDItemPixelHeight " & ¬
101 quoted form of thisImageFilePOSIXPath) as integer
102 return {imageWidth, imageHeight}
103 on error
104 return false
105 end try
106end getImageDimensions
