# WWDC 2003 Session 306: AppleScript Studio

**Speakers:** Sal Soghoian, Tim Bumgarner, John Coelho
**Year:** 2003
**Session:** 306
**Duration:** 56:30
**Index:** https://nonstrict.eu/wwdcindex/wwdc2003/306/

## Description

AppleScript Studio as a rapid application development environment — create standalone Cocoa applications using AppleScript. Drag-and-drop event handlers, Pasteboard class, data source enhancements, document-based application support, plug-ins for Xcode and Script Editor, Xcode scriptability.
