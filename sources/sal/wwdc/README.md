## How to get each session — the three tiers

### 1. The reliable source — nonstrict.eu/wwdcindex

Every session above has a `nonstrict.eu/wwdcindex/wwdc<year>/<num>/` page that contains:
- Title, speakers, duration, track, abstract
- Direct URLs to Apple's `devstreaming-cdn.apple.com` PDF and video assets (where Apple posted them)
- A **Whisper-generated transcript inline on the page** (older sessions; Apple-posted transcripts for newer ones)

Pattern: `https://nonstrict.eu/wwdcindex/wwdc<YEAR>/<SESSION>/`

### 2. Apple-posted PDFs

Pattern: `https://devstreaming-cdn.apple.com/videos/wwdc/<YEAR>/<HASHED-PATH>/<SESSION>/<SESSION>_<slug>.pdf`

The hashed path is unguessable; get the URL from nonstrict.eu. Captured so far:
- `2014-session-306/306_javascript_for_automation.pdf` (204 pp, 2.9 MB)

Apple never posted PDFs for WWDC 2003-2010 (the era was video-only). PDFs become common after 2012.

### 3. Video access — four tiers

- **Tier A (public CDN MP4):** Modern WWDC sessions (2014+) have public HD/SD MOV URLs on `devstreaming-cdn.apple.com`. Direct `curl` works.
- **Tier B (public HLS):** Older sessions (2012, 2013) have HLS streams on `devstreaming-cdn.apple.com/videos/wwdc/<HASH>/<NUM>/ipad_c.m3u8`. Download with `ffmpeg -i <m3u8> -c copy output.mp4`.
- **Tier C (auth-gated):** Some sessions only have downloadable MOVs behind Apple Developer login. Requires logged-in browser session.
- **Tier D (not available):** WWDC 2003-2009 video was never republished on the modern CDN. **Whisper transcripts on nonstrict.eu are the only durable record.**

## What we learned — the synthesis

### The 1997 → 2015 arc

- **1997** — Sal joins Apple as Product Manager for AppleScript (named on stage in 2010 as "the last thirteen years")
- **2003** — Four-session WWDC explosion: AppleScript Studio, AppleScript Update, AppleScript for SysAdmins, AppleScript and QuickTime. The peer-to-Aqua doctrine is articulated.
- **2004** — Automator previewed. AppleScript+QuickTime sequel. Tiger.
- **2005** — Audience broadening to C/C++/Java devs. Build-your-app-scriptable pitch.
- **2007** — Title shifts to "Product Manager, Automation Technologies." Leopard. Scripting Bridge ships → JavaScript on the OSA bus 7 years early. Automator 2.0.
- **2008** — Snow Leopard preview. Automator frameworks (`AMWorkflow`) validated by Savant Systems home automation.
- **2009** — Point-of-Need doctrine articulated. Four C's. AppleScriptObjC launches. macosxautomation.com soft-launched on stage.
- **2010** — iPad pivot. Content-pipeline doctrine: automation moves upstream when devices are sealed.
- **2011** — AppleScript-ObjC on the desktop without Xcode. Terminal joins the four-pillar canon.
- **2012** — Bridge talk. Automation survives sandboxing via Apple Event Access Groups + NSUserScriptTask + user-placed-file-as-consent.
- **2013** — 20th anniversary. AppleScript Libraries (`use` clause + .sdef authoring). Two paired sessions.
- **2014** — JXA launches. JavaScript as peer on OSA. `$` ObjC bridge with `registerSubclass` fulfills 2003's `call method` vision.
- **2015** — Enterprise focus. Accidental-administrator doctrine. Apple Configurator 2 + Automator. Sal's last WWDC.
- **2016** — Silence. Sal eliminated in October.
- **2018** — WWDC #717 announces Workflow → Shortcuts. Sal not invited.

### The four (later five) pillars — Sal's structural canon

1. **AppleScript** (1993)
2. **Automator** (2004)
3. **Services** (2009 redesign)
4. **Terminal** (2011 induction)
5. **JXA** (2014, the JavaScript peer)

This is the load-bearing structure for probing future Apple OS versions: which pillar has been removed, neutered, or relocated?

### Where the legacy lives in 2026

- **AppleScript:** still scriptable in Sequoia. SDEF format unchanged.
- **Automator:** present but de-emphasized; replaced in user-facing comms by Shortcuts.
- **Services:** still functional via `~/Library/Services/`.
- **Terminal:** still scriptable, working-directory tracking still works.
- **JXA:** still works via `osascript -l JavaScript`. Apple has not deprecated it.
- **QuickTime Player Pro:** dead. Modern QuickTime Player X has a drastically stripped sdef vs. QT Player 7 Pro.
- **`call method`:** still works in AppleScript Studio successors.
- **AppleScript-ObjC:** still works in `.applescript` applets.
- **`~/Library/Application Scripts/`:** still active. Still consent-by-placement.
- **`use` clause + Script Libraries:** still active.
- **Speakable Workflows:** retired; **Vocal Shortcuts** is the 2024 successor (memory anchor).
- **Apple Configurator 2:** still active; Automator action support retained.

### Why the archive matters

The 2003 transcripts are **the earliest primary-source spoken Sal in the entire WWSD canon** — they predate every interview source by 9 years. The 2007-2015 sessions document the structural maturation of Mac automation as a discipline: from one-pillar AppleScript (2003) to five-pillar canon (2014), with consent-by-placement (2012) and the JXA bridge (2014) as the architectural deltas.

**Where the archive doesn't yet reach:** 21 inferred sessions (2003-2009) listed above, where Sal was almost certainly present but isn't in the speaker list. Future passes can scrape those nonstrict.eu pages, identify which ones actually feature Sal-spoken content vs. which were team-member-only sessions, and extend the canon further.
