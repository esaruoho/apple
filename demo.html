Every Script Teaches

-- Toggle macOS dark mode on/off
-- App: System Events
-- Generated by workflow-gen.py

-- Concept: tell application ... end tell
-- Sends Apple Events to an app.
-- Concept: not (boolean toggle)
-- Inverts a boolean value.

tell application "System Events"
  tell appearance preferences
    set dark mode to not dark mode
  end tell
end tell
