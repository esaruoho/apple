# Sal Analysis

This directory stores notes derived from Sal source archives.

Subdirectories:
- `site-maps/` for navigational maps of each mirrored domain
- `page-notes/` for page-by-page summaries, extraction notes, and open questions
- `concept-notes/` for reusable teachings such as UI scripting, data passing, script menus, and accessibility

Rules:
- Keep quotations attributed with source URLs
- Do not mix normalized script code into these notes
- Link every note back to `sources/sal/<site>/manifest.yaml` and the relevant source page
