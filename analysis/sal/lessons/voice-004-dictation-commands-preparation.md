# Voice 004: Dictation Commands Preparation

Track: Accessibility And Voice
Level: 3
Category: Accessibility Preparation

## Source

- Site: `dictationcommands`
- Page: `sources/sal/dictationcommands.com/mirror/wayback-20240424/web.archive.org/web/20240424181821/http:/dictationcommands.com/preparation.html`

## Goal

Teach the setup and environmental prerequisites behind usable voice automation.

## What This Lesson Allows

- Show that voice workflows depend on system configuration
- Connect accessibility settings to automation reliability
- Explain why preparation is part of workflow design

## Repo Pairing

- Workflow: `scripts/workflows/accessibility/ax-system-settings-keyboard.applescript`

Why this pairing:
It keeps the learner focused on the settings layer, which is exactly where Dictation Commands preparation begins.

## Suggested Teaching Flow

1. Read the preparation page.
2. List the settings and prerequisites voice automation needs.
3. Run the keyboard-settings accessibility workflow.
4. Discuss how fragile automation becomes when its prerequisites are not explicit.

## Practice

- Compare `scripts/workflows/accessibility/ax-system-settings-keyboard.applescript` with `scripts/workflows/accessibility/ax-system-settings-privacy.applescript`.
- Describe a checklist for preparing a machine for voice automation.

## Modern Equivalent

- Type: `repo-workflow`
- Target: `scripts/workflows/accessibility/ax-system-settings-keyboard.applescript`

## Outcome

The learner understands that accessibility automation starts with environment setup, not just command execution.
