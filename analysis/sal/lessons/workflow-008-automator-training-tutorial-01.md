## Goal

Introduce Automator through Sal's training sequence rather than only the overview page.

## What This Lesson Allows

- Teach an end-to-end Automator exercise with training context
- Show how workflows are assembled from actions
- Connect the conceptual overview to direct manipulation in Automator

## Repo Pairing

- Workflow: `scripts/workflows/automator/automator-new-workflow.applescript`

Why this pairing:
It opens the tool itself and makes Automator a live environment rather than a historical concept.

## Suggested Teaching Flow

1. Read the first Automator training tutorial.
2. Explain actions, inputs, and outputs.
3. Run the Automator new-workflow script.
4. Compare building visually in Automator with writing direct AppleScript.
