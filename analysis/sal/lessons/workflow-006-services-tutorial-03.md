## Goal

Continue the Services sequence with a tutorial centered on transformation workflows.

## What This Lesson Allows

- Show how a workflow can turn one kind of input into a new output
- Teach transformation as a design pattern rather than a one-off script trick
- Reinforce the value of contextual tools embedded in user workflows

## Repo Pairing

- Workflow: `scripts/workflows/safari/safari-search-web.applescript`

Why this pairing:
It starts with a current term or concept and turns it into a new action in context, which matches the transformation mindset of Services.

## Suggested Teaching Flow

1. Read the third Services tutorial page.
2. Identify the input, transformation, and output stages.
3. Run the Safari search workflow.
4. Compare this transformation model with a pure inspection workflow.

## Practice

- Compare `scripts/workflows/safari/safari-search-web.applescript` with `scripts/workflows/safari/safari-current-url.applescript`.
- Write down a text-to-text or text-to-app transformation service you would actually use.
