# Modern 002: Dictionaries As Curriculum Scaffolding

Track: Modern Translation Layer
Level: 4
Category: Workflow Modernization

## Source

- Site: `repo`
- Page: `dictionaries/`

## Goal

Use extracted dictionaries to connect Sal's pedagogy to machine-readable app structures.

## What This Lesson Allows

- Move from page-based teaching into object-model exploration
- Show how dictionary references can act as curriculum support material
- Connect documentation, examples, and runnable scripts into one teaching loop

## Repo Pairing

- Workflow: `dictionaries/keynote/keynote.md`
- Runnable Example: `scripts/workflows/keynote/keynote-slide-count.applescript`

Why this pairing:
The dictionary explains what objects exist. The slide-count workflow shows how those objects become a concrete command.

## Suggested Teaching Flow

1. Open the Keynote dictionary material.
2. Identify the objects and commands relevant to slide counting.
3. Run `scripts/workflows/keynote/keynote-slide-count.applescript`.
4. Compare dictionary-first learning with page-first tutorial learning.

## Practice

- Open `dictionaries/numbers/numbers.md` and identify a comparable count-style command.
- Explain how a dictionary page helps you write a new script instead of only copying an old one.

## Modern Equivalent

- Type: `dictionary-plus-workflow`
- Target: `scripts/workflows/keynote/keynote-slide-count.applescript`

## Outcome

The learner sees how the repo's dictionary corpus can extend Sal's tutorial method into a modern, machine-readable exploration workflow.
