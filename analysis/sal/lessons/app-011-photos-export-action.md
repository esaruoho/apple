# App 011: Photos Export Action

Track: App Automation
Level: 3
Category: Photos Automation

## Source

- Site: `photosautomation`
- Page: `sources/sal/photosautomation.com/mirror/wayback-20240721/web.archive.org/web/20240721142125/http:/photosautomation.com/export-action.html`

## Goal

Deepen the Photos curriculum around export as a reusable action family.

## What This Lesson Allows

- Show how photo selections become files outside the library
- Connect media curation to downstream file-based workflows
- Reinforce the idea of output actions inside Photos

## Repo Pairing

- Workflow: `scripts/workflows/photos/photos-export-selected.applescript`

Why this pairing:
It is the current repo counterpart to the preserved export-action material.

## Suggested Teaching Flow

1. Read the export-action page.
2. Explain which properties matter when exporting from Photos.
3. Run the export-selected workflow.
4. Compare export with favorite and slideshow actions.

## Practice

- Compare `scripts/workflows/photos/photos-export-selected.applescript` with `scripts/workflows/photos/photos-favorite-selected.applescript`.
- Describe which workflows stay inside Photos and which move assets out of Photos.

## Modern Equivalent

- Type: `repo-workflow`
- Target: `scripts/workflows/photos/photos-export-selected.applescript`

## Outcome

The learner sees export as the bridge from library organization to external reuse.
