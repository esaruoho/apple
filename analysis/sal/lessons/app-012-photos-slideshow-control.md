# App 012: Photos Slideshow Control

Track: App Automation
Level: 3
Category: Photos Automation

## Source

- Site: `photosautomation`
- Page: `sources/sal/photosautomation.com/mirror/wayback-20240721/web.archive.org/web/20240721142125/http:/photosautomation.com/slideshow-control.html`

## Goal

Teach presentation-style control workflows inside Photos.

## What This Lesson Allows

- Show that Photos automation includes display and playback control
- Connect curation to presentation
- Expand the notion of media automation beyond metadata and export

## Repo Pairing

- Workflow: `scripts/workflows/photos/photos-start-slideshow.applescript`

Why this pairing:
It gives a visible, immediate action that matches the preserved slideshow-control material.

## Suggested Teaching Flow

1. Read the slideshow-control page.
2. Explain how display control differs from file output control.
3. Run the slideshow-start workflow.
4. Compare slideshow control to Keynote playback control.

## Practice

- Compare `scripts/workflows/photos/photos-start-slideshow.applescript` with `scripts/workflows/photos/photos-stop-slideshow.applescript`.
- Describe a use case where Photos is the better presentation surface than Keynote.

## Modern Equivalent

- Type: `repo-workflow`
- Target: `scripts/workflows/photos/photos-start-slideshow.applescript`

## Outcome

The learner sees that Photos can be automated as both a library and a presentation environment.
